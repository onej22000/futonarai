フトン洗いの匠

サイト概要
布団を洗えるクリーニング店「フトン洗いの匠」のサイト

サイトテーマ
フトン洗いの匠のサービス、予約、問い合わせなどができる。

テーマを選んだ理由
私はクリーニング店を経営しており、サービスの紹介や予約管理ができるサイトを作りたかったため。

ターゲットユーザ
布団を洗たい人、洗ってみたいが料金やサービスをまず調べてみたい人
